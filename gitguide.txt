GitGUIDE
/////////
https://help.github.com/articles/adding-an-existing-project-to-github-using-the-command-line/

1.Create a new repo in GitHub
2.Open GitBash
3.Change current working directory to your local project
4.Initialize local directory as a Git Repository
5.Add files to local repository
6.Git commit -m
7. Find URL of remote repository
8.git remote add origin "remote repository URL"
9.git remote -v *Login screen will show up"
10. git push origin master
